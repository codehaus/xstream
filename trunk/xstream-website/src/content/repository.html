Anonymous Subversion Access

This XStream source can be checkout out anoymously from this Subversion repository:

svn://svn.xstream.codehaus.org/xstream/scm/trunk/xstream

Example

svn co svn://svn.xstream.codehaus.org/xstream/scm/trunk/xstream

Developer Subversion Access

For commit access, developers require an SSH account.

svn+ssh://svn.xstream.codehaus.org/scm/xstream/trunk/xstream
svn+ssh://svn.xstream.codehaus.org/scm/xstream/trunk/xstream-website
