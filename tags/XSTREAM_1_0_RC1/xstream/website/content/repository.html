Anonymous CVS Access

This project's CVS repository can be checked out through anonymous using these details:

Method pserver
Server cvs.codehaus.org
Username anonymous
Password (empty)
CVS Root /scm/cvspublic
Module xstream

Example

cvs -d :pserver:anonymous@cvs.codehaus.org:/scm/cvspublic login
cvs -z3 -d :pserver:anonymous@cvs.codehaus.org:/scm/cvspublic co xstream
